Website Project: "Safari" Web App

This project showcases a complete web app I built from scratch, featuring user registration, login, personalized dashboard, and interactive pages for each service.

Key Features

  • Registration & Login System: Allows new users to register and returning users to log in securely.
  • Account Dashboard: Displays user-specific options, such as balance, payment, and package management.
  • Interactive Service Pages: Each service has its own page with detailed information and imagery.
  • Responsive Design: Optimized for seamless viewing on desktop and mobile devices.

Technologies Used

HTML, CSS, JavaScript, LocalStorage for user data handling, and custom media queries for responsive design.
